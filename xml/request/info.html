Parameters:
  • query:
    • cql-query, A Gentle Introduction to cql
  • source:
    • source of data (currently not supported)
  • facets:
    • number and list of facets
  • format:
    • dkabm, marcxchange or docbook
  • start:
    • First record counting from 1
  • stepValue:
    • number of records
  • sort:
    • date_ascending, date_descending, creator_ascending, creator_descending, random, rank_general, rank_title, rank_creator
  • agency:
    • (currently partially supported)
  • collectionType:
    • manifestation or work (defaults to work)
  • allObjects:
    • returns all objects in a collection
  • allRelations:
    • return relations for each objects
  • relationData:
    • return relation as "type", "uri" or "full"
  • outputType:
    • json, php, xml
  • callback:
    • for json-callback
